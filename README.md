# Space Tycoon

Супер новая и современная игра про космическую станцию (_дополненное описание_, кнопочку "развернуть" самостоятельно находим). Подробнее [ниже](#spacetycoon)

## Структура проекта

```
Assets/                                     # место хранения всех частей игры
    Prefabs/                                # сохранённые сложные переиспользуемые объекты
    Scenes/                                 # сцены (зачастую разные для разных тестов)

    Scripts/                                # папка со всеми C# скриптами проекта
        CameraController/                   # группа скриптов для работы камеры
        ModulesScripts (unattachable)/      # папка со вспомогательными скриптами для модулей

    Settings/                               # настройки проекта: input bindings, graphics
.gitignore
README.md                                   # вы находитесь здесь)
```

## Подготовка к работе

Выполняется с помощью одного из программистов. Можно писать либо в группу, либо напрямую этим людям (тг): @i_d_polyakov, @AlbionVisual
